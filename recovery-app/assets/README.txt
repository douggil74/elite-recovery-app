ASSET PLACEHOLDERS
==================

This directory should contain the following image assets:

1. icon.png (1024x1024)
   - App icon for iOS and Android
   - Should be a simple, recognizable icon
   - Suggested: Shield or badge icon

2. splash-icon.png (512x512)
   - Splash screen icon
   - Centered on splash background

3. adaptive-icon.png (1024x1024)
   - Android adaptive icon foreground
   - Should have transparent areas for adaptive backgrounds

4. favicon.png (48x48)
   - Web favicon

To create placeholder icons:
1. Use a simple shape (circle, shield, etc.)
2. Export at required sizes
3. Use #1a1a2e as background color where needed

Recommended tools:
- Figma (free)
- Sketch
- Adobe Illustrator
