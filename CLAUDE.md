# CLAUDE.md

## Project Overview

Elite Recovery of Louisiana - Fugitive recovery service website with dark, mysterious theme.

## Tech Stack

- **Next.js 16** with App Router
- **React 19**, **TypeScript**
- **Tailwind CSS 4**

## Development Commands

```bash
npm run dev      # Start dev server
npm run build    # Production build
npm run lint     # Run linter
```

## Subdirectory: Recovery App

The `/recovery-app` folder contains a separate Expo/React Native mobile app for recovery agents.
See `/recovery-app/ARCHITECTURE.md` for details.
